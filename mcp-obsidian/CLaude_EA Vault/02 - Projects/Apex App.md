# Apex — Personal Productivity App

← [[../WORK]]


Personal productivity web app with tasks, ideas, projects, calendar with time blocking, and Obsidian two-way sync.

## Architecture
- Next.js 16 App Router frontend
- Supabase (PostgreSQL) backend
- Local Node.js Obsidian sync agent
- Fixed left sidebar layout — 5 main pages

## Pages
- Dashboard
- Tasks (kanban board)
- Ideas (grid)
- Projects (list + detail)
- Calendar (time blocking)

## Key Features
- Two-way Obsidian sync
- Drag-and-drop tasks
- Supabase real-time data
