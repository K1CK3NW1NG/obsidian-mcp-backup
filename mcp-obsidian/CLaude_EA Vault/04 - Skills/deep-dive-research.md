# Deep-Dive Research Skill

Replaces 4–5 hours of manual research with a single invocation. Searches YouTube, X/Twitter, Reddit, GitHub, blogs — then saves a MASTER.md + source files to a dated folder.

## Phases
1. **Brainstorm** — Map the topic, generate angles, identify research focus
2. **References** — Analyze any provided links
3. **Broad Expansion** — Search YouTube, X, Reddit, GitHub, blogs
4. **Synthesize** — Write MASTER.md + source files

## Output
Saves to `research/YYYY-MM-DD-topic-slug/` in Claude_EA.

## Invocation
```
/deep-dive <topic> [optional links]
```
