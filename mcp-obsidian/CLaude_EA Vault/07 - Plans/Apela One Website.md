# Apela One Website — Implementation Plan

Build a 7-page luxury marketing website for Apela One LLC with glassmorphism design, scroll-driven animations, parallax effects, and a live-looking interactive product demo.

## Tech Stack
- Vanilla HTML/CSS/JS — no framework
- Shared design tokens in `css/tokens.css`
- CSS keyframes + Intersection Observer
- DM Sans (Google Fonts)

## Pages
1. Home (index.html)
2. Services
3. Pricing
4. About
5. Contact
6. Blog listing
7. Blog post template
