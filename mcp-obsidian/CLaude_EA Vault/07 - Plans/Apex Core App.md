# Apex — Personal Productivity App Plan

Build a personal productivity web app (Apex) with tasks, ideas, projects, calendar with time blocking, and two-way Obsidian sync.

## Tech Stack
- Next.js 16 App Router
- TypeScript
- Tailwind CSS v4
- Supabase JS v2
- @dnd-kit (drag and drop)
- date-fns

## Pages
- Dashboard
- Tasks (kanban)
- Ideas (grid)
- Projects (list + detail)
- Calendar (time blocking + unscheduled sidebar)

## Key Integration
- Two-way Obsidian sync via local Node.js agent
